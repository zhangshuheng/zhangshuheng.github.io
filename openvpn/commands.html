1、centos查看路由表:  ip route show|column -t

2、windows查看路由表: route print

3、centos添加路由:   ip route add 10.9.0.0/16 via 10.8.0.1 dev tun0

4、openvpn启动重启命令:

        systemctl stop openvpn@server.service

        systemctl start openvpn@server.service

5、删除路由:  ip route del 10.15.150.0/24

6、创建客户端key:./build-key client

7、启动linux客户端： sudo openvpn --config client.ovpn

8、centos7 安装openvpn

	yum install epel-release

	yum install openvpn easy-rsa -y

	
	systemctl status openvpn@client.service

	
	systemctl enable openvpn@client.service
	

	systemctl start openvpn@client.service
