wget https://get.enterprisedb.com/postgresql/postgresql-9.6.1-1-linux-x64-binaries.tar.gz


解压缩:

tar zxf postgresql-9.6.1-1-linux-x64-binaries.tar.gz


创建系统用户

useradd postgres

修改目录权限

chown -R postgres ./pgsql
切换用户

su postgres


初始化数据库，-D是指定数据存贮目录

bin/initdb -D data


启动

bin/pg_ctl -D data/ -l logfile start

登陆

bin/psql -d postgres
